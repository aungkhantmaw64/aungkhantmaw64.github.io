# Logistic Regression

Logistic Regression is one of the popular machine learning algorithms that is easy to understand and implement from scratch. Understanding the logistic regression algorithm can also be helpful in understanding artificial neural networks (ANN) since it works quite similarly to how a single neuron work.
So, before we deep dive into the mathematical part of the algorithm, we need to approach it as a black box that accepts an array of values (features) that describes an object (training example) and predicts (inference) what that object is in terms of probability.
